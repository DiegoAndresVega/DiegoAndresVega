## Proyectos personales

### Dividi — gastos compartidos full-stack, en producción

Desarrollada de principio a fin: modelo de datos, API REST, tests, despliegue y cliente móvil.

- **[dividi-FastApi](https://github.com/DiegoAndresVega/dividi-FastApi)** · `Python 3.12` `FastAPI` `PostgreSQL` `SQLAlchemy 2.0` `Docker` — 4 métodos de división de gastos, balances, simplificación automática de deudas y registro por invitación. 86 tests. **En producción** (Render + Neon): [dividi-api.onrender.com/docs](https://dividi-api.onrender.com/docs)

### [Costura](https://github.com/DiegoAndresVega/costura) — app Android de patrones de costura

`Kotlin` `MVVM` `Room` `Firebase` — proyecto final DAM: calcula la tela necesaria en patrones de costura y los comparte en comunidad.

### [Web de evento (Finkafest)](https://github.com/DiegoAndresVega/finkafest-web) — web interactiva en tiempo real

`React 18` `JavaScript` `Firebase Realtime Database` `Canvas API` `Cloudflare`
